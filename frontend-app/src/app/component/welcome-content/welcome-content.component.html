Boilerplate app

Spring Boot - Angular 4 app with authentication.

Email confirm and password reset are both implemented, and handled with an (async) email sending microservice.

Database maintence for deleting user also has a default implementation, and it is also decoupled from the app to a separate microservice.

Currently uses PostgreSQL, though it is easy to change, as Spring JPA handles the database.

Authentication is handled with Json Web Tokens.
